# Optimizing Garbage Collecting Routes in Barangay Virgen Delas Flores

### Mathematical Modeling FCO - BSM CS 3A

## Project Overview

This project applies Graph Theory and Mathematical Modeling to solve a Chinese Postman Problem for the Barangay Virgen Delas Flores, Baliuag, Bulacan Garbage Collection Routes using Ant Colony Optimization.

## How to use

Open `AntColonyOptimizationModel.ipynb` using Jupyter Notebook.

The algorithm uses SageMath to be able to draw figures such as the Graphs of the study.
